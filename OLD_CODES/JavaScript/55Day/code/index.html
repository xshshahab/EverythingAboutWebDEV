Variables and Data Types

Variables Declaration Example

var a = 10;

var b = 20;

var c = a + b;

console.log(c); // Output: 30

Data Types Examples with typeof Operator

  • Number: var num = 42;
  • String: var text = "Hello World";
  • Boolean: var isTrue = true;
  • Undefined: var notDefined;
  • Null: var emptyValue = null;
  • Object: var person = { name: "John", age: 30 };
  • Array: var colors = ["red", "green", "blue"];
  • Function: function sayHello() { return "Hello"; }
