Advanced CSS Overflow Property Examples

Overflow: Visible (Default)

The content overflows outside the box because `overflow: visible;` allows it to extend beyond the container's bounds.

Overflow: Hidden

The content is clipped and hidden. It does not scroll, and you cannot see any overflow.

Overflow: Scroll

The container shows scrollbars even when not strictly necessary.

Overflow: Auto

Scrollbars appear only when the content overflows the box.

Horizontal Overflow Only (`overflow-x: scroll`)

Scroll horizontally to see the hidden content.

Vertical Overflow Only (`overflow-y: scroll`)

Scroll vertically to see the hidden content.
