CSS Overflow Property Examples

Overflow: Visible (Default)

This is a demonstration of the `overflow: visible` property. The content can overflow outside the box boundaries.

Overflow: Hidden

This is a demonstration of the `overflow: hidden` property. The overflowing content is clipped and hidden.

Overflow: Scroll

This is a demonstration of the `overflow: scroll` property. A scrollbar appears even if all the content is visible within the box. Scroll to see hidden content.

Overflow: Auto

This is a demonstration of the `overflow: auto` property. A scrollbar appears only when the content exceeds the box size.
