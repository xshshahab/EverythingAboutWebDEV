HTML Entities and Code Tag

1. HTML Entities

HTML entities are used to represent special characters in HTML that would otherwise be interpreted as HTML code. For example:

  • < for less-than symbol (<)
  • > for greater-than symbol (>)
  • & for the ampersand symbol (&)
  • " for double quotes ("")
  • © for the copyright symbol (©)

Here’s an example of some HTML entities in action:

<div>This is a div element</div>

When rendered, the above code will show:

<div>This is a div element</div>

2. The <code> Tag in HTML

The <code> tag is used to define a piece of computer code. It’s typically used to represent programming languages, commands, or other code snippets.

Here is an example of code being used inside the <code> tag: 

        let x = 5;
            let y = 10;
            let sum = x + y;

Notice how the <code> tag helps in distinguishing the code from the rest of the text, maintaining the original formatting and style.
